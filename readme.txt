The project should be open in IntelliJ IDEA.

The program uses two initial prameters:
1. name of file with initial data(sample.in)
2. name of file with fees(fees.in)
The program can be run with no params, with sample only, or with both sample and fees.
For each mod there is related predefined run configuration within IDEA project.

! Files must not contain trailing new line at the end of file.
